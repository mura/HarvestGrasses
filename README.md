# Harvest Grasses Mod

## 概要
草を破壊した際に、種を落とす代わりにランダムにアイテムをドロップします。

## インストール方法
[Minecraft Forge](https://files.minecraftforge.net/maven/net/minecraftforge/forge/index_1.13.2.html)をインストール後、modsフォルダにjarを配置する。

## 動作確認環境
- Minecraft 1.13.2
- Minecraft Forge 25.0.84

## 謝辞
